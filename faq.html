Frequently Asked Questions

  • How do I add RAVEN to my server?

    You can add RAVEN to your server using this link

  • How do I change the prefix?

    You can change RAVEN's prefix with .setprefix [PREFIX]

  • RAVEN won't respond to any commands, what can i do?

    It's likely that you're using the wrong prefix, you can ping @RAVEN to get the prefix he's using and try running them again.
    If RAVEN still isn't responding to commands with the correct prefix then you should enable error messages with .errors enable

    Also check if he has all required permissions (send messages, embed links).

  • How can I report Bugs?

    1. Join the Support Server
    2. Open a new ticket in the support channel
