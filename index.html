Logging system

Has anyone trolled on your server? With the logging system of the RAVEN you can see who it was.

  • You can set up logging for different events:
  • You can set separate logchannels for each event, so that everything remains clear and you can find what you are looking for.
  • Read messages even if they have been edited or deleted
  • See when someone has joined, left or moved a voice channel
  • All logging messages are sent from a webhook that is created by RAVEN to spare the discord api

Join Role, Join Message, Leave Message

Automate your discord server with a join role and customizable join and leave messages. The setup is very simple

  • Different variables allow you to set the join and leave message the way you want it!
  • You can always view and change your current settings.

Temporary Voice Channels

The days with hundreds of unused voice channels are over! Keep your server clean.

As admin you can set a default channel limit. The users can change the limit, rename their channel and make it private or public.

Single users can also be permitted to join the voice channel or be rejected to join the channel

Ticket System

The ticket system with an archive function offers you the possibility to process the tickets of the users optimally.

  • Customizable ticket message when a new ticket gets created.
  • Customizable DM message when the ticket gets closed.
  • With the support role your server mods can manage the tickets without extra permissions.

F.A.Q.

Frequently Asked Questions

  • How do I add RAVEN to my server?

    You can add RAVEN to your server using this invite link

  • How do I change the prefix?

    You can change RAVEN's prefix with .setprefix [PREFIX]

  • RAVEN won't respond to any commands, what can i do?

    It's likely that you're using the wrong prefix, you can ping @RAVEN to get the prefix he's using and try running them again.
    If RAVEN still isn't responding to commands with the correct prefix then you should enable error messages with .errors enable

    Also check if he has all required permissions (send messages, embed links).

  • How can I report Bugs?

    1. Join the Support Server
    2. Open a new ticket in the support channel

    You can upvote RAVEN using this upvote link
